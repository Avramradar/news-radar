# NEWS RADAR

Автоматический Telegram-бот для канала `@newsRadar2026`.

## Что делает

- получает свежие материалы из RSS;
- оценивает важность по ключевым словам;
- исключает повторы;
- публикует короткие посты в Telegram;
- работает непрерывно на Railway.

## Запуск на Railway

1. Загрузите все файлы проекта в корень GitHub-репозитория.
2. В Railway создайте проект из этого репозитория.
3. Добавьте переменные:

```env
BOT_TOKEN=новый_токен_от_BotFather
CHANNEL=@newsRadar2026
POLL_MINUTES=20
POSTS_PER_CYCLE=2
```

4. Убедитесь, что бот является администратором канала и имеет право публиковать сообщения.
5. Нажмите Deploy.

## Важно

Ранее опубликованный в чате токен необходимо отозвать через BotFather и заменить новым.
